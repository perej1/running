# running

Here I simply collect my running data to the file `run.csv`. Explanations of the
variables are the following.

- `year`: Year of the running day (*2026*, *2027*, ...).

- `month`: Month of the running day (*1*, *2*, ..., *12*).

- `day`: Day of the month (*1*, *2*, ..., *31*).

- `weekday`: Day of the week (*mon*, *tue*, ..., *sun*).

- `distance`: The running distance (*km*).

- `speed_avg`: Average speed (*min.sec/km*).

- `bpm_avg`: Average heart rate (*beats per minute*).

Most of the time, the data for the variables `distance`, `speed_avg` and
`bpm_avg` are collected from my Polar Vantage M3 watch.
