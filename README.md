# running

Here I simply collect my running data to the file `run.csv`. Explanations of the columns are the following.

- `year`: Year of the running day.

- `month`: Month of the running day (*1*, *2*, ..., *12*).

- `day`: Day of the month.

- `weekday`: Day of the week (*monday*, *tuesday*, ..., *sunday*).

- `distance`: The running distance (*km*).

- `speed_avg`: Average speed (*min/km*).

- `speed_max`: Maximum speed (*min/km*).

- `bpm_avg`: Average heart rate (*beats per minute*).

- `bpm_max`: Maximum heart rate (*beats per minute*).
